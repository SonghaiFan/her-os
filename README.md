# Her OS

Modern Next.js rebuild of the archived `index.html` experience, tuned for a Vercel-style React setup.

## Stack

- Next.js App Router
- React 19
- TypeScript
- CSS with no additional UI runtime dependencies

## Run

```bash
npm install
npm run dev
```

The app uses the archived Samantha audio clips from `public/audio` and falls back to text input when browser speech recognition is unavailable.
